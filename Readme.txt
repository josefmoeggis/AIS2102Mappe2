1. Download Visual Studio Code if you do not already have it.

2. Make sure python is correctly installed on your device.

3. Open vscode and go to extensiont. Search for "Python" and install it.

4. Open the QUBE folder by pressing file -> open folder -> .../QUBE

5. Install the required libraries by copying and running: "pip install pyserial PyQt5 pyqtgraph"
   in a terminal. Simply press Terminal at the main menu in vscode and copy and paste.

6. Run main.py and pray

7. Also attempt to run gui.py
